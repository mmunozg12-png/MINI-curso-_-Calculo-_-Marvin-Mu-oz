Conceptos y definiciones

  • Sólido de revolución: Un sólido de revolución es un cuerpo tridimensional que se obtiene al girar una figura plana (llamada región generatriz) alrededor de una recta fija denominada eje de revolución. Este tipo de sólidos tiene simetría rotacional y su volumen se puede calcular utilizando métodos del cálculo integral, como los discos, arandelas o casquillos cilíndricos.
  • Región generatriz: La región generatriz es el área plana delimitada por una o más curvas (o rectas), que al ser rotada completamente (360°) alrededor de un eje, da lugar a un sólido de revolución. Es esencialmente la "forma base" que se transforma en un cuerpo tridimensional al girar.
  • Eje de revolución: El eje de revolución es la línea recta fija alrededor de la cual se gira la región generatriz para formar el sólido de revolución. Este eje puede estar dentro o fuera de la región generatriz, y su posición influye directamente en la forma final del sólido.

Fórmulas principales

Método de los discos (rotación alrededor del eje \(x\)):

\[ V = \pi \int_a^b [f(x)]^2 \, dx \]

Método de los anillos (hueco):

\[ V = \pi \int_a^b \left( [R(x)]^2 - [r(x)]^2 \right) dx \]

Método de las capas cilíndricas (rotación alrededor del eje \(y\), usando \(x\) como radio):

\[ V = 2\pi \int_a^b x\, f(x)\, dx \]

Fundamento teórico del cálculo integral en sólidos de revolución

Antes de calcular volúmenes de sólidos de revolución, es importante comprender el papel del cálculo integral. Una integral definida representa la suma continua de cantidades infinitesimales, lo que nos permite calcular áreas y volúmenes con precisión.

En el caso de los sólidos de revolución, la idea fundamental es imaginar que la región bajo una curva se divide en tiras muy delgadas. Al girarlas alrededor de un eje, cada tira forma un disco o un anillo, y la suma infinita de sus volúmenes da como resultado el volumen total del sólido.

La expresión general es:

\[ V = \lim_{n \to \infty} \sum_{i=1}^{n} \pi [f(x_i)]^2 \Delta x = \pi \int_a^b [f(x)]^2 dx \]

Esta fórmula es la base del método de los discos, y su extensión da lugar al método de los anillos y al de las capas cilíndricas, dependiendo del eje de rotación elegido.

Propuesta de Problemas

  1. Calcular el volumen al rotar \(y=x^2\) entre \(x=0\) y \(x=2\) alrededor del eje X.
  2. Determinar el volumen generado por \(y=\sqrt{x}\) entre \(x=0\) y \(x=4\) alrededor del eje X.
  3. Encuentra el volumen de la región limitada por \(y=x\) y \(y=2x-x^2\) al rotar alrededor del eje X.
Soluciones (opcional)

Problema 1: \(V=\pi\int_0^2 x^4 dx = \pi \left[\frac{x^5}{5}\right]_0^2 = \frac{32\pi}{5}\)

Problema 2: \(V=\pi\int_0^4 (\sqrt{x})^2 dx = \pi\int_0^4 x \, dx = \pi \left[\frac{x^2}{2}\right]_0^4 = 8\pi\)

Problema 3: Intersección: \(x=0\) y \(x=1\) (resolver \(x=2x-x^2\) -> \(x^2-x=0\) -> x=0,1). Luego \(V=\pi\int_0^1 ([2x-x^2]^2 - [x]^2) dx\)

Aplicaciones reales de los sólidos de revolución

Los sólidos de revolución no son solo un concepto matemático, sino una herramienta fundamental en múltiples campos de la ciencia y la ingeniería:

  • Ingeniería mecánica: Cálculo del volumen y peso de piezas rotacionales, como ejes, engranajes o pistones.
  • Arquitectura: Diseño de estructuras con formas cilíndricas, cónicas o esféricas (cúpulas, torres, columnas).
  • Medicina: Modelado 3D de órganos o vasos sanguíneos a partir de cortes bidimensionales.
  • Industria química: Determinación de la capacidad de tanques de almacenamiento y recipientes presurizados.
  • Informática y gráficos 3D: Generación de objetos tridimensionales mediante rotación de perfiles (en software CAD o Blender).

Estas aplicaciones muestran cómo el cálculo integral permite transformar conceptos abstractos en herramientas útiles para resolver problemas reales.

Programa de resolución (C++)

Programa que aproxima el volumen mediante la regla de Simpson (puedes cambiar la función f por la que necesites):

// Programa: aproximar volumen por método de Simpson
#include <iostream>
#include <cmath>
using namespace std;

double f(double x) {
    return x * x; // ejemplo: y = x^2
}

double volumen(double a, double b, int n) {
    if (n % 2 != 0) n++; // Simpson requiere n par
    double h = (b - a) / n;
    double suma = f(a)*f(a) + f(b)*f(b);
    for (int i = 1; i < n; i++) {
        double x = a + i * h;
        double fx2 = f(x) * f(x);
        if (i % 2 == 0) suma += 2 * fx2;
        else suma += 4 * fx2;
    }
    return M_PI * (h / 3.0) * suma;
}

int main() {
    double a, b;
    int n = 1000;
    cout << "Ingrese limite inferior (a): "; cin >> a;
    cout << "Ingrese limite superior (b): "; cin >> b;
    double V = volumen(a, b, n);
    cout << "Volumen aproximado: " << V << endl;
    return 0;
}

Notas: reemplaza la función f para resolver distintos ejemplos.

El propósito del código es calcular el volumen aproximado de un sólido de revolución, es decir, el cuerpo que se genera al girar la curva \(y=f(x)\) alrededor del eje X entre los límites \(a\) y \(b\).
